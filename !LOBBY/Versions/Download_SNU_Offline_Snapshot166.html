Version 3.x

SNU 3.x is the current branch of SNU, and currently the most modern. It takes all the ideas from the older versions, and combines them and adds on to them. The biggest difference is that it is written from the ground up, using several languages. It isn't made with Google sites, it isn't just some JavaScript files that don't work, it is a large functional database of files that you can go around in.

Development started on January 30th 2019 after successfully linking an HTML document to an HTML document. Sean went from there, and learned the rest of the neccessary web languages and kept building it into a powerful offline system that will one day go online.

This is SNU 3
